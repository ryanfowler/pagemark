# WCXB benchmark data

The WCXB data is not in this repository. The `VERSION` file gives the pinned source commit and archive SHA-256 value. Download that archive from the WCXB repository and verify its checksum before evaluation.

WCXB has a CC BY 4.0 license. Copyright belongs to its authors. See <https://github.com/Murrough-Foley/web-content-extraction-benchmark>.

Use the development set during routine work. Use the test set only for release review. Benchmark tooling is not part of the Pagemark command-line interface.
